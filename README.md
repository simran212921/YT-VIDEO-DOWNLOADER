# YouTube Video Downloader

## Description

This is a simple Python project that downloads YouTube videos using the yt-dlp library.

It is designed for beginners who want to learn how to:
- Take user input
- Use Python libraries
- Download files
- Handle errors
- Work with folders

## Features

- Download YouTube videos
- Saves videos automatically
- Creates downloads folder
- Easy to use

## Requirements

- Python 3.x
- yt-dlp

Install:

```bash
pip install -r requirements.txt
```

## How to Run

Run:

```bash
python main.py
```

Paste the YouTube video URL and press Enter.

The downloaded video will be saved inside the **downloads** folder.
